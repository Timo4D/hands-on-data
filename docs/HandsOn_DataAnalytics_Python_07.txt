Linkage methods
•
'single’: The distance between two nearest data  points (each point from a different cluster)
•
'complete’: The distance between two farthest data  points (each point from a different cluster)
•
'average’: The average of all pairwise distances between clusters
•
"centroid”: The distance between the centers (means) of the two clusters
•
"median”: The distance between the centers (medians) of the two clusters
•
"ward”: Minimizes increase in total within-cluster variance

Dimensionality reduction
Dimensionality reduction methods:
Techniques that transform high-dimensional data into a lower-dimensional representation 
while preserving as much relevant information as possible.
Motivations:
(1) Visualization
(2) Noise reduction
(3) Distilling patterns and insights
General principal:
•
Keeping important information: variance, relations, clusters
•
Discarding: noise, redundancy and irrelevant features

Principal Components Analysis (PCA)
Principal Components Analysis (PCA): 
A method to finds the axes (“principal components”; PCs) of maximum variance in the data,  
using linear combinations of the original axes (i.e. original features). 
Such that:
•
PC1 (first principal component): Represents the direction of maximum variance in the data
•
PC2 (second principal component): is the axis which is orthogonal (perpendicular) to PC1 
and which captures the maximum of the variance which is not already captured by PC1
•
… and so on with PC3, PC4…

Data standardization:
It is essential to perform data standardization before applying PCA
•
PCA is driven by variance
➢Data standardization equalizes the variance of the features, thus equalizes their influence
•
Centering of the data of each feature around 0 (as also achieved by the data standardization) 
eliminates offsets and thus ensures that the PCA will capture the true directions of the variance

t-Distributed Stochastic Neighbor Embedding (t-SNE)
t-SNE (t-Distributed Stochastic Neighbor Embedding):
•
A nonlinear dimensionality reduction algorithm
•
Focuses on preserving local structures in the data
•
Data points which are very similar to each other are kept together in a cluster
•
Distances between clusters are typically meaningless → so global structures are not well preserved

Main t-SNE parameters:
• n_components: Number of output dimensions (usually 2 or 3, for visualization)
• perplexity: Controls how many nearby points influence where point is placed (~5–50, must be < number of samples)
• Low values: focus very local structures → small, fragmented clusters
• High values: Consider a bit more also global structures
• learning_rate: Step size in gradient descent 
• n_iter / max_iter: Number of optimization iterations (~1000–3000)
