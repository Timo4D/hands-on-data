Input arguments in functions must appear in the follwoing order (from left to right):
1. Positional-only arguments
2. Positional or keyword arguments
3. *args   (variable positional arguments)
4.  Keyword-only arguments
5. **kwargs (variable keyword arguments)
Rules for the order of input arguments of mixed types with *args and **kwargs
Additional rules:
•
All parameters before “/” must be passed positionally
•
All parameters after the “*” must be passed using keywords
•
All required parameters without defaults must come before parameters with defaults

Mutability of objects in Python
Mutability: whether an object can be changed after it is created
Mutable objects can be modified in place.
➢Therefore, changes in such an object affect all the references to it. ()
Examplr for mutable obtects:
•
list
•
dict
•
set
Immutable objects cannot be modified.
➢Therefore, any “change” in it actually creates a new object.
Immutable obtects:
•
int
•
float
•
str
•
tuple
