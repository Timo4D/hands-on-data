Regression analysis:
A statistical method to model and estimate the relationship between a dependent variable and one or more 
independent variables.
•
Infers the direction (sign) of this relation and quantifies how strong it is
•
Identifies which independent variables are the most influential ones
•
Enables to predict the value of the dependent variable given values of the independent variables

Choosing the decision threshold
The decision which decision threshold to choose depends on our goal:
•
If false positives and false negatives are equally tolerable for our purpose: 
➢= 0.5 would be an optimal decision threshold
•
If we want to reduce the risk of missing real positives and can tolerate false positives 
(e.g. cancer screen test): 
➢ < 0.5 would serve this goal
•
If we want to reduce the risk of having false positives and can tolerate false negatives 
(e.g. identifying molecular markers and targets for diagnostics and therapy): 
➢ > 0.5 would serve this goal
