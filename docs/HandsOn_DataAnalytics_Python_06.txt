•
One-sample t-test: 
➢Tests if the population mean is different than an hypothesized mean
•
Independent two-sample t-test (Unpaired): 
➢Tests if the population means of the two independent groups are different
•
Paired t-test:
➢Tests if the mean difference between paired observations is not zero

FDR correction
False discovery rate (FDR) =   
Total number of false positives
Total number of positives (true+false) 
When testing multiple null hypotheses at once (e.g. when comparing multiple pairs of groups):
”Positive”: a case in which the null hypothesis gets rejected
”False positive”: a case in which the null hypothesis is rejected but the null hypothesis is actually true
”True positive”: a case in which the null hypothesis is rejected and the alternative hypothesis is indeed true

ANOVA
One-way ANOVA tests the null hypothesis that all groups have the same population mean
Null hypothesis (H₀):  
 
 
𝐻0: 𝜇1 = 𝜇2 = 𝜇3 = ⋯= 𝜇𝑛
𝜇1 = population mean of group I
𝑛 = total number of groups
Alternative hypothesis (H1):  
 
At least one group mean is different
The test does not identify which group is different
Assumptions:
•
Normal distributions
•
Independent observations
•
Similar variance

Post-hoc tests
In case ANOVA test / Kruskal-Wallis H-test reject the null hypothesis:
➢Post-hoc tests can be applied to identify which groups are different
➢This is done by performing pair-wise group comparison tests for all possible pair-wise 
combinations:
•
Mann–Whitney test (to follow Kruskal-Wallis H-test) 
•
T-test (to follow-up ANOVA)
