Mutability of objects in Python
Mutability: whether an object can be changed after it is created
Mutable objects can be modified in place.
➢Therefore, changes in such an object affect all the references to it. ()
Examplr for mutable obtects:
•
list
•
dict
•
set
Immutable objects cannot be modified.
➢Therefore, any “change” in it actually creates a new object.
Immutable obtects:
•
int
•
float
•
str
•
tuple
Variables are names (labels) that reference objects in memory
