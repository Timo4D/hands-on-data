Tabular data – a universal data structure
Raw data: The data in the form in which it got originally acquired 
Such data has the form of a table (hence „tabular data“): 
Data points (=“object”) X Features
•
Data point: an individual observed example (“object”) within the dataset. (e.g., a given house)
•
Feature: an individual property that describes each data point (e.g., price)
Most commonly, the data contains values for the features of objects
For example:
•
Objects:    Houses
•
Features:  area, location, price,… of each house
